# UN Bet - Front

## Para trabajar en este repo

0. Asegúrate de tener node y verifica el npm.
1. Hacer fork.
2. Crear Branch con el nombre de la funcionalidad.
3. Testear.
4. Hacer fix (si es necesario).
5. Volver al paso 3 hasta que esté lista la funcionalidad.
6. Hacer pull request.

## Docker

1. Asegúrate de tener docker instalado.
2. usar el docker-compose (docker-compose up) para montar.

##Verificando

1. Usa "npm install".
2. Usa el comando "npm run dev".
3. Si todo funcionó deberías tener un local montado para ver la página.
